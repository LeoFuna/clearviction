# Clearviction Subscription

> Form Subscription

## About

This project is a simple form to send a subscription with email and github url.

- [Used Tools](#used-tools)
- [Dependencies](#dependencies)
- [Starting](#starting)

## Used Tools

- Next JS
- Typescript
- Tailwind Css

## Dependencies

- Node > 18
- Next JS > 13

## Starting

Installing dependencies

```bash
npm install
```

Running project locally

```bash
npm run dev
```

src/
